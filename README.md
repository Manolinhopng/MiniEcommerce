## 🌟 Vision
**MiniEcommerce** isn't just a shop; it's a curated digital experience. Designed for brands that demand perfection, it combines lightning-fast performance with a level of visual polish usually reserved for high-end luxury apps.

## ✨ Elite Features

### 💳 Next-Gen Checkout Experience
Our checkout flow is designed to minimize friction and maximize trust:
- **Intelligent Card Detection**: Instant visual identification of Visa, Mastercard, AMEX, and Discover.
- **Luhn Algorithm Shield**: Real-time structural validation to prevent "fat-finger" errors before they happen.
- **Dynamic 3D-Style Card Preview**: Watch your card come to life as you type, providing immediate visual confirmation.

### 🎭 Cinematic User Interface
- **Motion-Sickness Proof**: Fluid, physics-based transitions powered by Framer Motion.
- **Glassmorphism Design**: Sophisticated translucent layers that adapt to your background.
- **Seamless Dark Mode**: High-contrast, easy-on-the-eyes dark theme that respects system settings.

### 🚀 Performance First
- **Zero-Layout Shift**: Optimized image handling and skeleton states.
- **React 19 Ready**: Leveraging the latest concurrent rendering features for a butter-smooth feel.
- **Vite 8 Engine**: Instant HMR and ultra-fast production builds.

## 🛠️ Technical Masterpiece

| Stack Component | Implementation Details |

## 🚀 Experience It Locally

### 1. Preparation
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Ignite the Project
```bash
# Clone the masterpiece
git clone https://github.com/Manolinhopng/MiniEcommerce.git

# Enter the vault
cd MiniEcommerce

# Install the engine
npm install
```

### 3. Configuration
Copy the blueprint to your local environment:
```bash
cp .env.example .env
```
*Note: Ensure `VITE_APP_NAME` is set to your desired store name.*

### 4. Lift Off
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) and witness the future of e-commerce.
